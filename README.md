# Steel City Codes Starter Projects

Beginner Python starter projects for command-line programming practice.

## How to Use This Repo

Each project folder contains:

- `README.md` with the project description, requirements, examples, and manual tests
- `starter.py` with incomplete starter code and TODO comments

Run a starter file from a terminal:

```bash
cd projects/magic_8_ball
python starter.py
```

Use `python3 starter.py` if your computer uses `python3` for Python 3.

## Recommended Order

Phase 1:

- Magic 8 Ball
- Rock Paper Scissors

Phase 2:

- Chatbot
- Tic-Tac-Toe
- Hangman

Phase 3:

- Connect 4
- Adventure Map

All projects are designed for terminal input and printed output.
